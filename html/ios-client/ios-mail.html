Configure iOS Mail

If you want to use iOS Mail with Ubikom, and you are viewing this page on your iPhone in Safari, you can use the button below to download the configuration profile.

The following link will allow you to download iOS Mail configuration profile. Click this link, and download the profile.

Install the Profile

After the profile is downloaded, you will need to go to Settings, General, VPN & Device Management. You will see the downloaded profile, tap on it and install it. You will be asked for your password. After the profile is installed, open iOS Mail and you will be ready to go.

Optional: Change Name

If you want to change your name (the name other users see when you send them an email), you can do so under Settings, Mail, Accounts. Choose the Ubikom account, tap on Name, and change it to whatever you prefer.
