Общая настройка клиента

Эта страница содержит инструкции по настройке Ubikom Mail на обычном клиент, то есть любой почтовый клиент, поддерживающий протоколы SMTP и IMAP. У большинства клиентов так.

Проверьте свои идентификационные данные

На странице идентификационных данных у вас должно быть как минимум следующее:

  • Ваш адрес электронной почты
  • Ваш пароль

Если ваш клиент не поддерживает автоматическую настройку, вам также понадобятся:

  • Исходящий адрес сервера и порт
  • Входящий адрес сервера и порт
  • Имя пользователя (такое же, как ваш идентификатор, адрес электронной почты также будет работать)

Создать новую учетную запись IMAP

Большинство почтовых клиентов поддерживают протокол IMAP. Когда вы создаете новую учетную запись, выберите IMAP (иногда вам нужно будет выбрать «другой» или «почтовый клиент» и т. д.)

Укажите данные учетной записи

Вам нужно будет заполнить данные учетной записи. Скорее всего, они придут через несколько группы, общую информацию об учетной записи, сведения о входящем сервере (IMAP) и исходящие сведения о сервере (SMTP). Давайте пройдемся по ним.

Общая информация об учетной записи

Здесь вам нужно будет указать несколько вещей:

  • Адрес электронной почты — это ваш идентификатор с добавленным к нему «@ubikom.cc». Если ваш идентификатор — «bob», ваш адрес электронной почты — «bob@ubikom.cc».
  • Имя учетной записи — это просто имя для вашей учетной записи. Что-то типа Подойдет "Убиком мэйл".
  • Пароль — клиент может использовать этот пароль для входа по IMAP (и, возможно, по SMTP).
  • Тип учетной записи — если вы можете выбирать между POP и IMAP, выберите IMAP.

Входящая информация о сервере

Здесь вы указываете информацию для вашего входящего (IMAP) сервера:

  • Адрес сервера - введите "imap.ubikom.cc""
  • Порт сервера - 993, что по умолчанию.
  • Имя пользователя — это ваше имя пользователя со страницы идентификационных данных.
  • Пароль - ваш пароль.
  • Безопасность - выбрал SSL/TLS.
  • Аутентификация — выберите Обычный или Обычный пароль.

Исходящая информация о сервере

Этот раздел предназначен для исходящего (SMTP) сервера:

  • Адрес сервера - введите "smtp.ubikom.cc"
  • Порт сервера — 465, что по умолчанию.
  • Имя пользователя — это ваше имя пользователя со страницы идентификационных данных.
  • Пароль - ваш пароль.
  • Безопасность - выбрал SSL/TLS.
  • Аутентификация — выберите Обычный или Обычный пароль.
  • Если у вас есть флажок, говорящий что-то вроде «сервер требует аутентификации», убедитесь, что он отмечен

Проблемы?

Если ваша установка не работает, перепроверьте свои настройки. Вы можете захотеть полностью удалите эту учетную запись и попробуйте настроить ее снова.

Проверьте список известных проблем здесь. Если вы считаете вы нашли ошибку, отправьте новую проблему, или свяжитесь по елецтронной почте email lgx@ubikom.cc.
